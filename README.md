# Step-by-Step Lists

Practicing the use of nested resources, as based on the [Rails Resources][1] guide.

##To get started with this repo:

`git clone https://github.com/vogelbek/nested_resources.git`

`cd nested_resouces`

`bundle install`

`rake db:setup`

`rails server`
